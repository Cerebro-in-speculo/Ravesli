Создайте перечисление со следующими перечислителями: chicken, lion, giraffe, elephant, duck и snake. Поместите перечисление в пространство имен. Объявите массив, где элементами будут эти перечислители и, используя список инициализаторов, инициализируйте каждый элемент с количеством лап (ног) определенного животного. В основной функции выведите количество ног слона, используя перечислитель.
